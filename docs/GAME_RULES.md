# Game rules

## Quick Hunt

1. Choose a seed parent, a pollen contributor, a project goal, and a run seed.
2. Inspect 12 deterministic seedlings. Cull at least one to make the first selection.
3. During vegetation, reserve at least one backup clone. Clone space and funds are limited.
4. Advance through five flower weeks and resolve the seeded garden event.
5. Choose an abstract harvest window and cure strategy. Each changes the finished trait field.
6. Run all five sensory steps: stem rub, jar aroma, grind aroma, flavor evaluation, and effect timeline.
7. Compare evidence, select one keeper, name it, and continue with an F2 or backcross.

The highest visible number is not automatically best. Traits remain hidden until the appropriate stage; goal fit measures alignment with the original project, while stability, linkage drag, sensory identity, and player taste create tradeoffs.

## Resources

- **Funds** pay for backup clones and some event responses.
- **Research** buys certainty or technical event responses.
- **Grow slots** cap the active population at 12.
- **Clone capacity** protects only three active candidates.
- **Reputation** records respected decisions and finished keepers.

## Controls and accessibility

All actions are available by keyboard. Important states use text and shape as well as color. Settings include interface scaling, audio volume/mute, reduced motion, three manual save slots, import/export, and a glossary. Irreversible actions ask for confirmation; the most recent cull can be undone.
